#+title: SQLite

SQLite est une version légère, tous est contenu dans un fichier. Elle est très utilisée.

La version courante est `sqlite3` et s'installe: ~sudo apt-get install sqlite3~

Les fonctions d'aide commence en général par un ~.~

L'URL typique ressemble à ~sqlite:///nom_db.db~


ex: ~.exit~ pour quitter
